# 🤖 TakwiniBot – Plateforme intelligente de recrutement..

## 📌 Overview

**TakwiniBot** est une plateforme web tunisienne intelligente dédiée à l'emploi des personnes en situation de handicap, développée dans le cadre du module **Projet Technologies Web (2A)** à *Esprit School of Engineering – Tunisia* (2025–2026).

Elle permet de connecter intelligemment les candidats en situation de handicap avec des entreprises inclusives grâce à :

* un système de **matching intelligent basé sur l’IA adapté aux besoins spécifiques**
* un accompagnement via un **coaching CV automatique**
* des **recommandations personnalisées**
* une prise en compte des **critères d’accessibilité (télétravail, aménagements, environnement de travail)**
